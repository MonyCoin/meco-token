## 💸 أنواع الرسوم الحالية

### 1. رسوم التبادل اللامركزي (Swap Fee)
- **الوصف:** رسم خدمة يضاف إلى كل عملية تبادل (Swap) عبر Jupiter API داخل التطبيق.
- **القيمة:** `0.0005 SOL` (ثابتة بغض النظر عن حجم الصفقة).
- **التنفيذ:** تُضاف هذه الرسوم تلقائياً كتعليمة تحويل منفصلة (SOL Transfer) ضمن معاملة التبادل على شبكة Solana.

### 2. رسوم عمليات التخزين (Staking Fee)
- **الوصف:** رسم خدمة يخصم عند تنفيذ عملية إيداع (Stake) أو سحب (Unstake) في شاشة MECO Earn.
- **القيمة:** `0.0005 SOL` لكل عملية.
- **التنفيذ:** تُضاف كتعليمة تحويل SOL ضمن معاملة التخزين/السحب.

## 🔄 آلية استخدام الرسوم

- **الغرض الأساسي:** تغطية تكاليف استمرارية التطوير، صيانة البنية التحتية (RPC nodes)، ودعم جهود التسويق المجتمعي.
- **الشفافية:** يمكن لأي شخص تتبع رصيد هذه المحفظة وتحركاتها بشكل مباشر عبر Solscan. يتم الإعلان عن أي استخدام رئيسي للرصيد عبر قنوات التواصل الرسمية للمشروع.
- **السياسة:** الرسوم **لا تُستخدم** لتحقيق أرباح شخصية للمؤسسين، بل تُعاد استثمارها بالكامل في نمو المنظومة.
