# Contributing to MECO

شكرًا لاهتمامك بالمساهمة في مشروع MECO!

## كيف تساهم؟
1. قم بعمل Fork للمستودع
2. أنشئ فرعًا جديدًا للتعديل
3. أرسل Pull Request
4. سيتم مراجعة التعديلات واعتمادها في حال كانت مناسبة

## قواعد بسيطة
- لا يتم قبول أي تعديل يمس عقود التوكن إلا بعد تدقيق
- التوثيق مرحب به دائماً
- الإضافات يجب أن تكون مفيدة ومباشرة
